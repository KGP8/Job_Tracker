GitHub Raw URL setup
Connect your db.json file hosted on GitHub so data persists across devices and sessions. When connected, the app reads from GitHub on load and you manually export + commit to save.
1
Create a GitHub repo (e.g. mrkgp-job-tracker) and push both index.html and db.json files.
2
Open db.json on GitHub → click Raw → copy the URL.
It looks like: https://raw.githubusercontent.com/username/repo/main/db.json
3
Paste the raw URL below and click Connect. App will load data from GitHub on every visit.
4
To save changes: click ↓ export → download updated db.json → commit to GitHub repo.
